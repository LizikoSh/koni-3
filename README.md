# Шляхетні — сайт про коней

Готовий статичний багатосторінковий сайт українською мовою. Не потребує збірки, npm або серверної частини.

## Структура

- `index.html` — головна сторінка;
- `breeds.html` — інтерактивний довідник із 6 розгорнутими профілями та алфавітним покажчиком 44 порід з української Вікіпедії;
- `stories.html` — особисті історії;
- `poetry.html` — тематична поезія;
- `assets/css/styles.css` — дизайн і адаптивність;
- `assets/js/main.js` — мобільне меню, пошук, фільтри й уподобання;
- `assets/images/` — локальні зображення.

## Публікація на GitHub Pages

1. Створіть новий репозиторій на GitHub.
2. Завантажте в корінь репозиторію весь вміст цієї папки.
3. Відкрийте **Settings → Pages**.
4. У полі **Source** виберіть **Deploy from a branch**.
5. Виберіть гілку `main`, папку `/ (root)` і натисніть **Save**.

Після публікації всі внутрішні посилання та зображення працюватимуть на GitHub Pages без додаткових налаштувань.
