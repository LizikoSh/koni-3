# Кінний Світ

Готовий статичний сайт українською мовою про коней.

## Сторінки

- `index.html` — головна;
- `breeds.html` — довідник порід із фільтром;
- `coats.html` — масті та відмітини;
- `terminology.html` — словник із пошуком;
- `facts.html` — цікаві факти;
- `stories.html` — художні й пізнавальні історії;
- `sport.html` — кінний спорт;
- `sources.html` — використані джерела.

## Запуск локально

Відкрийте `index.html` у браузері. Для перевірки через локальний сервер у папці проєкту можна виконати:

```bash
python -m http.server 8000
```

Після цього відкрийте `http://localhost:8000`.

## Публікація на GitHub Pages

1. Створіть новий репозиторій.
2. Завантажте в нього весь вміст цієї папки.
3. Відкрийте **Settings → Pages**.
4. Оберіть **Deploy from a branch**, гілку `main`, папку `/root`.
5. Збережіть налаштування.

## Зображення

Головне зображення підготовлене у форматах WebP і JPG у `assets/images/`.
